# Space Testing [![Circle CI](https://circleci.com/gh/meteor-space/testing.svg?style=svg)](https://circleci.com/gh/meteor-space/testing)

Helpers for Testing Space Applications and Modules.

## Install
`meteor add space:testing`

## Run the tests
`mrt test-packages ./`
